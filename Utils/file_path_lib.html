# glob/globby

glob 是一个 python 下的文件操作相关模块，用其可以查找符合要求的文件，支持通配符操作。

globby 是一个基于fast-glob的库，其在之上添加了一些有用的特性，如：Promise API, 扩展目录foo -> foo/**/*。

一个简单的 js 示例：

import {globby} from 'globby'
let matchedArr = await globby(['*', '!cake'])
console.log(matchedArr)
