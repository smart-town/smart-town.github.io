# sed 命令

> 参考 [原文](https://www.cnblogs.com/along21/p/10366886.html)

## 简介

命令格式:`sed [options] '[position] command' [file(s)]`

## 命令详解

### options

- `-n`: 只打印匹配到的行
- `-e`: 多点编辑
- `-r`: 支持扩展的正则表达式
- `-i`: 直接将处理结果写入文件
- `-i.bak`: 处理结果写入文件之前备份
- `-f`: 将 Script 写入文件,执行时指定文件路径

### position

- 省略,则对全文处理
- 单地址: 
    - `#` 指定行
    - `/pattern/` 被此模式匹配的行
- 地址范围:
    - `#,#`, `/pat1/,/pat2/`
    - `#,+#`
    - `#,/pat1/`

### command

- `d`: 删除模式空间匹配的行
- `p`: 打印模式空间的内容,追加到默认输出之后
- `a`: 在指定行后面追加文本,支持`\n`
- `i`: 在行前面插入文本,支持`\n`
